# Zushar V2

> Form building tool

![zushar logo](https://github.com/shevigroup/zushar/blob/master/src/assets/logos/zushar_1.png)

## Build Setup

**Uses node v20+

``` bash
# install dependencies
pnpm install

# serve with hot reload at localhost:8080
pnpm dev

# build for production with minification
pnpm build
```
